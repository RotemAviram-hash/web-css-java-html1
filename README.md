## 💻 הפרויקטים שלי - בריף טכני

### 1. 📝 רשימת מטלות (To-Do List)

- **דגש טכני:** מימוש **Data Persistence** באמצעות `LocalStorage` לשמירת מידע בדפדפן וניהול מצב (State) של משימות בזמן אמת.
- **טכנולוגיות:** HTML5, CSS3, JavaScript ES6, LocalStorage.

### 2. 🎮 איקס עיגול (Tic-Tac-Toe)

- **דגש טכני:** פיתוח **Game Logic** לבדיקת מצבי ניצחון ותיקו, ניהול תורות שחקנים ומניעת דריסת נתונים ב-DOM.
- **טכנולוגיות:** JavaScript ES6, CSS Grid, Event Handling.

### 3. 🧠 משחק הזיכרון (Memory Game)

- **דגש טכני:** ניהול אלגוריתם להשוואת זוגות, מימוש שלוש רמות קושי דינמיות וניהול אינטראקציות משתמש מורכבות.
- **טכנולוגיות:** JavaScript DOM Manipulation, CSS Animations.

### 4. 🧮 לומדת מחשבון (Math Trainer)

- **דגש טכני:** יצירת **Random Question Generator** המותאם לרמת המשתמש, אימות קלט (Validation) ומתן פידבק מיידי.
- **טכנולוגיות:** JavaScript Logic, CSS Flexbox.

### 5. 🌤️ אפליקציית מזג אוויר (Weather Dashboard)

- **דגש טכני:** אינטגרציה עם **REST API** חיצוני, עבודה אסינכרונית עם **Async/Await** והטמעת מפות אינטראקטיביות (**Leaflet.js**).
- **טכנולוגיות:** Fetch API, Leaflet.js, JSON.

### 6. 🌍 מנוע חיפוש מדינות (Country Explorer)

- **דגש טכני:** אחזור נתונים מ-API גלובלי, מימוש מנגנוני סינון (Filtering) מתקדמים ושימוש ב-**RegExp** להדגשת תוצאות חיפוש.
- **טכנולוגיות:** REST API, JavaScript ES6, Regular Expressions.

### 7. 💡 Smart Quiz (Trivia App)

- **דגש טכני:** ניהול זרימת נתונים אסינכרונית, מימוש טיימרים (**Timers**) ומערכת ניהול "חיים" ופסילות מבוססת לוגיקה.
- **טכנולוגיות:** REST API, JavaScript Timers, CSS Grid.

## 📥 הורדה והרצה מקומית (Installation & Setup)

כל הפרויקטים בפורטפוליו זמינים להורדה ולשימוש מקומי.

### אפשרות א': הורדה ישירה מהאתר

1. לחצו על כפתור **"הורדה"** בכרטיס הפרויקט הרלוונטי בפורטפוליו.
2. חלצו את קובץ ה-ZIP שהורד למחשבכם.
3. פתחו את קובץ ה-`index.html` בדפדפן.

### אפשרות ב': הרצה מ-GitHub

1. בצעו `Clone` למאגר: `git clone [כתובת המאגר שלך]`.
2. פתחו את התיקייה ב-Visual Studio Code.
3. להרצה אופטימלית, מומלץ להשתמש בתוסף **Live Server**:
   - התקינו את התוסף **Live Server** מ-VS Code Marketplace.
   - לחצו קליק ימני על קובץ ה-`index.html`.
   - בחרו ב-**"Open with Live Server"**.
   - האתר ייפתח בכתובת המקומית `http://127.0.0.1:5500`.
